# Final Project Overview

## Project Summary

This repository is organised for a Durham University Master-level final project combining SQL and Python. The goal is to demonstrate strong database design, data engineering, analysis, and reporting skills through a reproducible project workflow.

## Repository Structure

- `CODE/`
  - Contains Python scripts, Jupyter notebooks, SQL query files, and application logic.
  - Recommended contents:
    - `data_preparation.py` or `notebooks/data_preparation.ipynb`
    - `sql/` for schema definitions, queries, and stored procedures
    - `analysis/` for exploratory work, charts, and modelling
    - `utils/` for reusable helper functions and database connectors

- `DATA/`
  - Contains raw and processed data used for the project.
  - Recommended contents:
    - `raw/` for original source files
    - `processed/` for cleaned and transformed datasets
    - `samples/` for example data extracts or smaller subsets

- `Presentation/`
  - Contains assessment deliverables and documentation.
  - Recommended contents:
    - final slide deck
    - written report or executive summary
    - supporting figures, diagrams, and evaluation notes

   - findings, charts, and summary tables
